# Service Details: AI Marketing Agent (n8n + Outreach)

## Core Value Proposition
- **Automated Lead Generation**: Scrapes and processes 5,000+ leads monthly.
- **n8n Outreach Engine**: Dual-stack automation using Cold Email (SMTP/Ollama) and LinkedIn (Playwright).
- **AI Research**: Automated competitor research and meeting "Battle Cards" using local Ollama (Llama 3.1).
- **Region-Specific**: Optimized for both International and Ethiopian (local) markets.
- **A/B Testing**: Built-in variant testing to optimize reply rates.
