# Gumroad Copywriter

This skill specializes in crafting persuasive, concise, and conversion-focused product descriptions specifically for Gumroad's platform.

## Workflow

1. **Understand the Service**: Read [references/SERVICE_DETAILS.md](references/SERVICE_DETAILS.md) to understand the technical and business value of the AI Marketing Agent.
2. **Select Copy Framework**:
   - Use **PAS (Problem-Agitation-Solution)** for founders struggling with manual outreach.
   - Use **Feature-Benefit** for technically-minded agencies.
   - Use **Outcome-Focused** for high-ticket sales teams.
3. **Generate Output**: Produce a complete Gumroad listing including:
   - Catchy Title
   - Short Description (Summary)
   - Detailed Description (Rich text style)
   - Key Features List
   - Call to Action (CTA)

## Guidelines

- **Tone**: Professional yet approachable ("Senior Engineer who understands Sales").
- **Platform Style**: Avoid excessive fluff. Use bolding and lists. Use emojis sparingly but effectively.
- **Conversion focus**: Emphasize the **$500/mo value** and the **5,000 lead scale**.
- **Local Context**: Mention the Ethiopian/International split optimization where relevant to show deep market understanding.
